## 🚀 How to Run

**Option 1 — Open directly:**
```
Double-click index.html in your file manager
```

**Option 2 — Local server (recommended for best experience):**
```bash
# Python 3
cd portfolio
python -m http.server 8080
# → Open http://localhost:8080

# Node.js (npx)
npx serve .
```

## 🛠 Tech Stack

- **Pure HTML5, CSS3, Vanilla JavaScript** — zero dependencies
- **Intersection Observer API** — scroll-based animations
- **requestAnimationFrame** — smooth cursor & particle animations
- **Canvas API** — particle system
- **LocalStorage** — theme & dev mode persistence
- **CSS Custom Properties** — full theme system
- **CSS Grid & Flexbox** — responsive layout
